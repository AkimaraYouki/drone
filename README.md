## 프로젝트 개요

- **목표**: PyBullet 시뮬레이터에서 드론 모델을 띄우고,
1축(고도) 제어를 위해 **PID, 퍼지 로직, 강화학습** 3가지 기법을 순차적으로 구현·튜닝·비교
- **가치**:
    - 제어 이론(클래식·퍼지) 간 장단점을 직접 체험
    - 문헌 조사 → 구현 → 실험 → 분석 → 보고서 작성의 완전 연구 사이클 체험
    - Python 생태계(PyBullet, scikit-fuzzy, Matlab) 활용
